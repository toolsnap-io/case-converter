# Case Converter Tool

A free online tool to convert text between different letter cases instantly.

## Supported Conversions

- **UPPERCASE** - Convert text to all caps
- **lowercase** - Convert text to all small letters
- **Title Case** - Capitalize the first letter of each word
- **Sentence case** - Capitalize only the first letter
- **camelCase** - For programming variable names
- **snake_case** - For Python-style identifiers
- **kebab-case** - For URLs and CSS classes
